`Azure EventHub integration for New Relic Logs` offers an integration to forward log messages from [Azure EventHub](https://azure.microsoft.com/en-us/services/event-hubs/) to [New Relic Logs](https://docs.newrelic.com/docs/introduction-new-relic-logs). This allows you to collect logs from all your Azure services (except App services) and view them in [New Relic Logs](https://docs.newrelic.com/docs/introduction-new-relic-logs).

<Link to="https://github.com/newrelic-experimental/newrelic-azure-logs-integration">Installation Instructions</Link>

## Features

- Writes Azure EventHub data to New Relic Logs
